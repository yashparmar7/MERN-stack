What is Bootstrap ?

- Bootstrap is a free and open-source front-end framework for building responsive and mobile-first web applications. It provides a set of pre-designed HTML, CSS, and JavaScript components, as well as a grid system, that can be used to create a consistent and visually appealing user interface.

-"Bootstrap is a powerful,feature-packed frontend toolkit. Build anything-form prototype to production-in minutes."
