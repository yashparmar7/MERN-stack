Alpha Channel :- 

- Set the opacity for a color
- Range from 0 to 1
- Default value is 1.0
  
  rgba (255,255,255,1) 

Opacity :-

- Sets the opacity for the element
- Range from 0 to 1
- Default value is 1.0


Transition in CSS :- 

- Transition enable you to define the transition between two states os an element.

Shorthand :- proprty name | duration | timing-function | delay

ex :- transition : margin-top 2s ease-in-out 0.2s;



Transform in CSS  :-

-This property lets you rotate,scale,skew,or translate an element.

1) Rotate :-
- Rotate an element around a specified point.
    transform : rotate(45deg);

2) Scale :-
- Scale an element up or down.
    transform : scale(2,3);

3) Translate :-
- Move an element from its current position.
    transform : translate(50px,100px);

    transform : translateX(50px);
    transform : translateY(100px);

4) Skew :-
- Skew an element along the x-axis or y-axis.
    transform : skew(30deg);

Position property :-
-it's set how an elements is positioned in a document.
- The top,right,bottom and left properties determine the final location of positioned elements.

Static  :- the top,right,bottom,left and z-index properties have no effects.This is Default value. 
Relative :- The offset is relative to itself based on the values of top,right,bottom and left.
Absolute :- The element is removed from the normal document flow, and no space is created for the element in the page layout. It is positioned relative to its closest positioned ancestor, if any : otherwise , it is placed relative to the intial containing block.
Fixed    :- The element is removed from the normal document flow, and no space is created for the element in the page layout. It is positioned relative to the initial containing block established by the viewport.
