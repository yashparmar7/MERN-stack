For Loop :-
Used to repeat a piece of code.

syntax :-

for(initialization, condition, updation){
    //code
}



Nested Loop :-

syntax :-

for(let i=1; i<=3; i++){
    for(let j=1; j<=3; j++){
        console.log(j);
    }
}


While Loop :-

syntax :-

while(condition){
    //code
    //updation
}

ex :-

let i = 1;

while(i<=5){
    console.log(i);
    i++;
}


Break Keyword :-
- Stop the Loop Execution.
